Package calculadora

Class Calculadora

Esta classe fornece operacoes matematicas basicas como soma, subtracao, multiplicacao e divisao.

Metodos:

public int somar(int a, int b)

Soma dois numeros inteiros.

Parameters: a - primeiro operando, b - segundo operando

Returns: o resultado da soma

public int subtrair(int a, int b)

Subtrai dois numeros inteiros.

Parameters: a - primeiro operando, b - segundo operando

Returns: o resultado da subtracao

public int multiplicar(int a, int b)

Multiplica dois numeros inteiros.

Parameters: a - primeiro operando, b - segundo operando

Returns: o resultado da multiplicacao

public int dividir(int a, int b)

Divide dois numeros inteiros.

Parameters: a - primeiro operando, b - segundo operando

Returns: o resultado da divisao

Throws: IllegalArgumentException - se b for zero

public int calcular(int a, int b, String operador)

Executa a operacao matematica solicitada com base no operador informado.

Parameters: a - primeiro operando, b - segundo operando, operador - simbolo da operacao

Returns: o resultado da operacao

Throws: IllegalArgumentException - se o operador nao for valido
